# Dr. Alves de Sousa - Website Institucional

Website desenvolvido para a Clínica Dentária Dr. Alves de Sousa em Viana do Castelo.
Focado em performance, SEO local e conversão de leads.

## Stack Tecnológica
- **Framework:** Next.js 14 (App Router)
- **Styling:** TailwindCSS
- **UI:** Lucide React, Framer Motion
- **Features:** Chatbot de Árvore de Decisão, Marcação via WhatsApp/Formulário.
